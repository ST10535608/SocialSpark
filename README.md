# SocialSpark

## Description
SocialSpark is a Android application developed using Kotlin in Android Studio. The app was created to help Cora, who is struggling to maintain social connections during busy days. The app suggest quick and meaningful social interactions based on the time of day, making it easy and enjoyable to stay connected. 

## Features 
- Suggests social sparks based on time of day
- Morning: Send a Good Morning message to a family memeber
- Mid-morning: Reach out to a colleague with a quick Thank you
- Afternoon/Snack Time: Send a quick thinking of you message
- Dinner: Call a friend or relative for a 5 minute catch-up
- After Dinner/Night: Leave a thoughtful on a friend's post
- Reset button to clear input and suggestion
- Error handling for invalid input 

## Design considerations 
- Dark navy background for a sleek modern look
- Gold and white colour scheme for contast and making it easy to read
- Sun and moon icons to represent time of day
- Clean and simple layout for easy navigation

## GitHub Actions 
Github Actioins was used to automatically build the app everytime the code is pushed to the respository. This is to make sure the app builds successfully and not locally but also on the GitHub server

## How to Use 
1. Open the app on your Android device
2. Type in the time of day (e.g. Morning, Mid-Morning, Afternoon, Afternoon Snack Time, DInner, After Dinner/Night)
3. Press Get Suggestion to receive your social spark
4. Press New Time of Day to rest and try again
